# WuWa IT

Traduzione italiana amatoriale per Wuthering Waves.

## Come usare

1. Scarica il launcher da [WuWa ITA Launcher](https://github.com/AndreaVizz/wuwa-IT/releases)
2. Avvia il launcher e seleziona la cartella del gioco
3. Il launcher scaricherà e installerà automaticamente la traduzione

In alternativa, scarica manualmente il file `.pak` e copialo in `Client\Content\Paks\` nella cartella di Wuthering Waves.
